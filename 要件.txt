目的
スタッフの希望休・希望勤務日数・スキルバランスを考慮し、1時間ごとのシフトを自動作成する。

機能要件
• スタッフ情報管理
• スキル4項目（メイン、サブ、ホール、リーダーシップ）
• 希望勤務日数
• 希望休登録
• 18歳未満フラグ登録（21:00以降シフト禁止）
• シフト設定管理
• 1時間単位の必要人数設定
• 5項目のスキル目安点数設定
• シフト自動作成
• 希望休を絶対守る
• 必要人数を満たす
• スキル目標にできるだけ近づける
• 希望勤務日数にできるだけ合わせる
• 18歳未満の夜間勤務禁止
• シフト結果表示
• 作成したシフトの画面表示
• CSVまたはExcelでダウンロード（オプション）

非機能要件
• Webブラウザから操作
• スマホ対応（最低限の表示崩れ防止）
• 管理者ログイン制御（将来的に追加予定）

技術スタック
• バックエンド：Python + FastAPI
• データベース：MySQL
• フロントエンド：HTML/CSS（最初はシンプルなフォーム）
• ORM：SQLAlchemy
• 最適化ライブラリ：PuLP
• 実行環境：Docker + Docker Compose
• 開発環境：VS Code

その他
• 最初はローカル開発環境のみ
• 将来的にRender等で無料公開を検討
